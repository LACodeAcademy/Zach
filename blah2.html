It is good practice to put all of your standard elements on the page (divs, h1, text, etc) then have an embedded style sheet at the top of the page to do the styling. The reason is because later on when your boss says, "I want all tables to have a red background" you can alter code in one place in the heading and all tables on the page change color. It is really crazy what you can do with CSS. There's a website called csszengarden.com. Go there. What they do is provide web developers with a basic html page. The developers can't alter any of the html. All the developer does is create a CSS sheet which styles the html. So on the page there are links on the right side. Every link is to a different developer's page. The html is exactly the same on ALL pages, the only difference is the css which completely makes the page look different. Same content, different style.
